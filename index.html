About

Minecraft Sexy is a service that allows users to request and consume media using services such as Emby, Sonarr, Radarr, Jackett, and Ombi. This page will allow me to communicate the status of the service, along with some information regarding questions people may have about the service. Some information about each service that users can interact will follows.

Emby

Emby is the front-end of the whole operation that allows you to watch and download content from the host server.

Ombi

Ombi is a simplified way of requesting media. This software has been somewhat unstable in the past.

Why isn't my request being fulfilled? #

There are a number of reasons that a request isn't being fulfilled...

  • I ran out of space to download new requested media. (happens a lot)

How do I log in? #

As of now, the log in process varies between devices. I've tried to cover the main devices here.

  • Browsers/Computers
  • Phones
  • Consoles/TVs
Browsers/Computers #
Currently there is an Emby app for Windows, but I recommend the web client if you aren't interested in downloading media and would be fine just streaming.
  1. Go to app.emby.media
  2. Click Next
  3. Enter your Username/Email and Password and click Sign In.
  4. If everything went well, you should be at a screen like this. Click Minecraft Sexy or whatever server you want to browse.
  5. Done! From there, you can browse media and watch stuff.
Phones #
Following the steps above would also work on your phone, since you're essentially going to a website through a web browser. However, if you would like to use an app, these steps will get you there.
  1. Download the Emby from the app store on your phone.
  2. Open the app and tap Next
  3. Enter your Username/Email and Password and click Sign In.
  4. You should be a screen like this one. Click on the server you want to browse.
  5. Done! Go crazy.
Consoles/TVs #
Not all TVs will have an Emby app available to install. If yours does not, but you have a web browser available, I recommend going to tv.emby.media and skipping to Step 2.
  1. Download the Emby app onto your TV and open it.
  2. Hit Next
  3. You should see a screen similar to this one. Simply follow the directions on the screen.
    1. Take note of the PIN code on the screen
    2. On another device, go to emby.media/pin
    3. Enter your Email/Username, Password, and finally, the PIN code that is on your TV's screen
  4. Done!
